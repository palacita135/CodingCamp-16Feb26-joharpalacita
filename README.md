# Premium To-Do List Application

A modern, responsive, and aesthetic To-Do List web application built with HTML, CSS, and JavaScript.

## Features

- **Premium UI**: Dark mode, glassmorphism effects, and smooth animations.
- **Task Management**: Add, delete, and toggle completion status of tasks.
- **Filtering**: Tabbed interface to view All, Pending, or Completed tasks.
- **Persistence**: Tasks are saved locally in the browser (localStorage).
- **Validation**: Prevents adding empty tasks or tasks without a date.

## Technologies Used

- **HTML5**: Semantic structure.
- **CSS3**: Custom styling, variables, flexbox, and animations.
- **JavaScript (ES6)**: DOM manipulation and state management.
- **Font Awesome**: Icons.
- **Google Fonts**: Inter font family.

## Setup Instructions

1.  Clone this repository.
2.  Open `index.html` in your web browser.
3.  Enjoy organizing your tasks!
